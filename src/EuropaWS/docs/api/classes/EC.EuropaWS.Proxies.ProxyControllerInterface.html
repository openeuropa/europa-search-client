\EC\EuropaWS\Proxies

\EC\EuropaWS\ProxiesProxyControllerInterface

Interface ProxyControllerInterface.

Implementing this interface allows objects to control the conversion of a Message to send the web service into a format usable by it and call the transporter implementation for sending the result.
