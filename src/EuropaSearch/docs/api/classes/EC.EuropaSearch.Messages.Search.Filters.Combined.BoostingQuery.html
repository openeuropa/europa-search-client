\EC\EuropaSearch\Messages\Search\Filters\Combined

\EC\EuropaSearch\Messages\Search\Filters\CombinedBoostingQuery

Class BoostingQuery.

Represents a Boolean query compound of Europa Search; I.E: "Allows you to bias some of the fields, positively or negatively. Unlike the boolean query, even if the field does not match it will still be returned. This is just for the ranking of the results."

It only supports "Value" and "Values" objects that involve "StringMetadata" or "NumericMetadata".
